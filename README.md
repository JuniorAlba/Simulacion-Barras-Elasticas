Este repositorio contiene la implementación computacional para la resolución del Trabajo Práctico 1 de Mecánica del Continuo. El objetivo del proyecto es simular y analizar la respuesta dinámica de un sistema de partículas plano, vinculado mediante barras elásticas, que se encuentra inicialmente en reposo y es sometido a una carga uniforme en el tiempo. El código desarrollado está configurado específicamente para resolver la geometría y condiciones de borde correspondientes al Caso 12.
